# 🌍 SouthVista Travels – Travel Website

## 📌 Project Overview

**SouthVista Travels** is a prototype travel website designed to showcase popular tourist destinations across **South India**, especially **Karnataka**.
The website presents travel destinations, tour packages, multimedia content, and travel information to give users an engaging browsing experience.

This project demonstrates the use of **HTML elements, multimedia integration, and webpage structuring** to create a simple tourism website.

---

## ✨ Features

* 🏞 Image gallery of famous tourist destinations
* 📍 Embedded Google Maps location
* 🎥 Travel experience videos from YouTube
* 🎧 Travel-themed audio clips
* 📋 Travel information tables (bus schedules and tour packages)
* 🧭 Lists of destinations, services, and tour packages

---

## 🛠 Technologies Used

* **HTML5**
* **Basic CSS (internal styling)**
* **Google Maps Embed**
* **YouTube Video Embed**
