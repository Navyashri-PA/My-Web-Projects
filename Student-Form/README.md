# 🎓 Student Details Form – HTML Project

## 📌 Project Overview

The **Student Details Form** is a simple HTML-based web form designed to collect student information for academic or placement purposes.

The form includes sections such as personal details, contact information, education background, placement preferences, and account creation.

This project demonstrates the use of **HTML form elements and webpage structuring**.

---

## ✨ Features

* Student **personal information form**
* Contact and address details
* Educational qualification section
* Placement skills and domain selection
* Salary expectation and preferred job location
* Account creation fields
* Reset and submit functionality

---

## 🛠 Technologies Used

* HTML5
